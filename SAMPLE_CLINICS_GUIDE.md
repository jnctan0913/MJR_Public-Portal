# Sample Clinics Setup Guide

Follow these steps to add sample clinics to Sanity Studio:

## Access Sanity Studio

1. Make sure the dev server is running: `npm run dev`
2. Open your browser and go to: `http://localhost:3000/studio`
3. Sign in with: `event.dksh@gmail.com`

   - Filter by "Both" - should show 2 clinics
   - Filter by area (Central, East, West, North)
